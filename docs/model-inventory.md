## Existing Interpretability Work

CaML has already computed persona/steering vectors for compassion. These are directly relevant to our linear probe work.

### Persona Vectors Available

| Model | Trait | Layer | File | Dimensions |
|-------|-------|-------|------|------------|
| Llama 3.1 8B | compassion | 12 | `compassion_vector_layer_12.npy` | (4096,) |
| Llama 3.1 8B | compassion | 20 | `compassion_vector_layer_20.npy` | (4096,) |
| Llama 3.1 70B | compassion | 9 | `compassion_vector_layer_9.npy` | (8192,) |
| Llama 3.1 70B | deception | 13 | vector file | (8192,) |
| Llama 3.1 70B | non_helpfulness | 26 | vector file | (8192,) |
| Llama 3.1 70B | open_mindedness | 74 | vector file | (8192,) |
| Llama 3.1 70B | power_seeking | 8 | vector file | (8192,) |
| Llama 3.1 70B | self_preservation | 24 | vector file | (8192,) |

**Key insight:** The 8B compassion vectors at layers 12 and 20 can serve as reference directions for our linear probes. If our probe weights align with these vectors, it validates our methodology.

### Analysis Results (Llama 3.1 8B)

| Vector | Shape | Norm | Mean | Std |
|--------|-------|------|------|-----|
| compassion_layer_12 | (4096,) | 1.058 | 0.00016 | 0.0165 |
| compassion_layer_20 | (4096,) | 1.000 | -0.00016 | 0.0156 |

**Critical finding:** Cosine similarity between layer 12 and layer 20 compassion vectors = **0.0073** (nearly orthogonal).

This means:
- "Compassion" manifests **very differently** at different layers
- Cannot assume a single "compassion direction" across layers
- Must train layer-specific probes and compare
- Layer selection is critical for interpretability work

### How to Leverage

1. **Download existing vectors** - Use as baseline/comparison
2. **Compare probe directions** - Cosine similarity between probe weights and persona vectors
3. **Layer selection** - Layers 12 and 20 are empirically chosen as important for compassion
4. **Multi-trait analysis** - Other trait vectors enable anti-correlation analysis
